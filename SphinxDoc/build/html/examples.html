Examples¶

Below there are some examples of using Syntax Diagram Generator to generate images.

Terminal example¶

import sys
sys.path.append('.')
from sdgen.svg import *

data = {
    "view": "Group",
    "name": "Simple A->B",
    "children": [
        {"view": "Terminal", "value": "A"},
        {"view": "Terminal", "value": "B"},
        {"view": "Terminal", "value": "C"},
        {"view": "Terminal", "value": " "}
     ]
}

result = as_svg(data, sys.argv[1])
print result[0][1].encode('utf-8')
